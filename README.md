# MaisLeituraSite é um site realizado pelos alunos do terceiro informática a-vespertino, o intuito é realizar um cartalogo digital da biblioteca Alan Figueiredo para melhor acesso tanto dos alunos quanto dos professores.


# Acervo da biblioteca, utilizado para pesquisa de livros nos quais estão disponíveis na biblioteca, e, além disso, irá compartilhar notícias do CETEP da Bacia do Paramirim com o uso de banner's. Será usado não só para pesquisa, mas irá oferecer informações sobre a biografia do livro e do autor, visando aumentar ainda mais o repertólio dos alunos, extendendo tamném o interesse daqueles que irão ler.
